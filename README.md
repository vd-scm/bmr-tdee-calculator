# BMR & TDEE Calculator (Java)

A Java console application that calculates:
- Basal Metabolic Rate (BMR)
- Total Daily Energy Expenditure (TDEE)
- Target caloric intake based on deficit or surplus
- Daily macronutrient targets (protein, carbs, fat)
